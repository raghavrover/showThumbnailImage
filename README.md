### Project Funcitonalties

<details>
<summary>Functionality added</summary>
<br/>

The app has the following functionalities

- Initially, the first image in the list will be displayed
- When the user clicks on a thumbnail, then the corresponding image will be displayed
- The `Gallery` component has a `imagesList` array. It consists of a list of image details objects with the following properties in each object

  |       Key        | Data Type |
  | :--------------: | :-------: |
  |        id        |  Number   |
  |     imageUrl     |  String   |
  |   thumbnailUrl   |  String   |
  |   imageAltText   |  String   |
  | thumbnailAltText |  String   |

</details>
